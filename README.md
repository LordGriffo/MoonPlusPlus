# MoonPlusPlus
Animation made with C++ and SFML

---

# Dependencies

+ [gcc](https://gcc.gnu.org/)/[g++](https://gcc.gnu.org/)
+ [GNU Makefile](https://www.gnu.org/software/make/)
+ [SFML](https://www.sfml-dev.org/)

---

# Building

```sh
git clone https://github.com/LordGriffo/MoonPlusPlus
cd MoonPlusPlus
make
```
